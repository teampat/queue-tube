Everything you need

➕

Add to Queue

An "Add to Queue" button appears on every YouTube video — home feed, search, channels, sidebar, and Shorts.

▶️

Auto Play

Automatically plays the next video in the queue when the current one ends. Toggle on/off anytime.

✅

Auto Remove

Finished videos are removed from the queue automatically, keeping your list clean.

↕️

Drag & Drop

Reorder your queue by simply dragging items up or down to your preferred order.

🔖

Tab Tracking

Queue playback always continues in the same browser tab — no jumps, no confusion.

🎛️

Popup UI

Manage your queue from the extension popup with Now Playing strip, prev/next controls, and badge count.

Install manually

  1. Download or clone this repository from GitHub
  2. Open chrome://extensions/ in your browser
  3. Enable Developer mode (toggle in the top-right)
  4. Click Load unpacked
  5. Select the queue-tube folder
